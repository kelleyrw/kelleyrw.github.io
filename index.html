/accounts

Collection of available accounts in RevOne.

/accounts get post

get

Get a list of accounts.

post

Add a new account to RevOne.

get /accounts

Get a list of accounts.

Secured by oauth_2_0

Data+Math supports OAuth 2.0 for authenticating all API requests.

  • Response
  • Security

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

[
  {"account_id" : 1, "account_name" : "foo"},
  {"account_id" : 1, "account_name" : "foo"}
]

Secured by oauth_2_0

Headers

  • Authorization: required (string)

    Used to send a valid OAuth 2 access token. Do not use with the "access_token" query string parameter.

Query Parameters

  • access_token: required (string)

    Used to send a valid OAuth 2 access token. Do not use with the "Authorization" header.
